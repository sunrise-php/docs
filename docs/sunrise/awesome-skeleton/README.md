# Awesome Skeleton

Awesome Skeleton for modern PHP 8.1+ development.

## System Requirements

PHP 8.1+

## Installation

```bash
composer create-project 'sunrise/awesome-skeleton:^4.0' awesome-app/
```

```bash
cd awesome-app/
```

## Running

```bash
php -S localhost:8000 public/index.php
```
